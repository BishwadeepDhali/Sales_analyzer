📊 Sales Data Analyzer

A simple Python project that records daily sales, performs basic analysis, and visualizes the data using line and bar graphs.

🚀 Features

- Validates numeric input using "try-except"
- Calculates highest, lowest, and average sales
- Displays both line and bar graphs using "matplotlib"

🛠 Tech Used

- Python 3
- Matplotlib

▶️ How to Run

Install matplotlib (if needed):

pip install matplotlib

Run the script:

python sales_analyzer.py

📌 Purpose

Built as a beginner data analysis and visualization project to practice Python fundamentals and structured programming.
